Frequently Asked Questions

Is Stylo AI completely free to use?

Yes, Stylo AI is currently free to use. This includes generating, editing, saving, and deploying websites. We rely on Google's Gemini API, which has its own usage limits, but for most users, it's more than enough for regular use.

Can I export the code and host it myself?

Absolutely! The "View Code" feature in the editor gives you direct access to the complete HTML, CSS, and JavaScript. You can copy and paste this code anywhere you like and host it on your own server or services like Netlify, Vercel, or GitHub Pages.

Who owns the websites I create?

You do. You own 100% of the code and the final website that you generate with Stylo AI. You have full rights to use it for personal or commercial projects.

How good is the AI-generated code?

The code is generated by Google's powerful Gemini model and is generally high-quality, modern, and responsive. It uses standard HTML5, CSS (often with Flexbox or Grid), and vanilla JavaScript. While it's very capable, we always recommend reviewing the code before using it in a critical production environment.
